# Contributing

Thank you for considering contributing to this educational demo!

This project was built as an **educational template** to help developers learn how to build compliant Real World Asset (RWA) tokens on Hedera EVM.

### How to Contribute

- **Bug reports & feature requests** → Open an Issue
- **Educational improvements** → Submit a Pull Request
- **New examples** (e.g. adding dividend payments, locking periods, etc.) are especially welcome

### Development Setup

```bash
git clone https://github.com/kbennett2000/hedera-first-rwa-dapp.git
cd hedera-first-rwa-dapp
npm install
cd frontend && npm install
```

See `docs/DEPLOYMENT.md` for deployment instructions.

Made with ❤️ as an educational resource for the Hedera ecosystem.
